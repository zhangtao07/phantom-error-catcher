Phantomjs 页面错误自动检测 
=========

## 说明

使用phantomjs自动跑页面列表检查页面初次执行的JS错误，协助FE进行前端页面测试。

使用场景：如国际化多国家页面测试

## 安装使用

 - 安装Node和PhantomJs环境
 - `npm install `安装模块
 - 在conf/config.js中按格式添加需要检测的页面列表
 - 执行`start.bat`或者`start.sh`
